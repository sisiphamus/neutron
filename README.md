# Neutron

A Next.js application for nuclear and particle physics simulations.

Neutron is a planned interactive environment for visualizing subatomic processes: radioactive decay chains, neutron scattering, fission/fusion dynamics, and particle interactions. The goal is to turn abstract physics into something you can manipulate and watch unfold in real time.

## Planned

- Radioactive decay chain simulator with configurable isotopes and half-lives
- Neutron scattering cross-section visualizer
- Nuclear fission chain reaction modeling
- Energy level diagrams and transition animations
- Interactive periodic table with nuclear properties

## Tech

Next.js 16, React 19, TypeScript, Tailwind CSS 4

## Status

Scaffolded and ready for implementation. Part of a computational science toolkit alongside [plasma](https://github.com/sisiphamus/plasma), [monte-carlo](https://github.com/sisiphamus/monte-carlo), [stat-arb](https://github.com/sisiphamus/stat-arb), and [black-scholes-lab](https://github.com/sisiphamus/black-scholes-lab).
