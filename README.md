# NEUTRON

We split the atom eighty years ago and most people still can't picture what actually happens inside a nucleus.

Neutron is an interactive environment for nuclear and particle physics -- decay chains unfolding step by step, neutron scattering visualized as it happens, energy level diagrams you can actually explore rather than memorize. Fission isn't an abstract concept here. You watch a neutron hit U-235 and follow the cascade.

The goal is to make the most powerful and most invisible force we've ever harnessed into something you can see and reason about intuitively.

Scaffolded and not yet functional. Built with Next.js + TypeScript.

```bash
npm install && npm run dev
```
